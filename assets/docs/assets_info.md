
# Assets / Stammdaten

Dieser Ordner enthält **keine Personendaten**.

## assets/data
- `baureihen.json` – Baureihen **nur als Codes** (z. B. `185`, `425`), alphabetisch/numerisch sortiert.
- `evu_bv.json` – EVU/BV **nur als Codes** (z. B. `MDW`, `SBBIN`).
- `pruefarten.json` – Prüfarten als Codes.
- `netzteile.json` – optional; aktuell leer.

## assets/examples
Beispiel-Payloads ohne echte Personendaten.

> Personendaten/Exports (Ausbildungsakten, externe DB) **nicht** ins öffentliche Repo committen.
